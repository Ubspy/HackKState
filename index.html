MewsicGen

Generates a random song -- first choose a method -- made for Hack K-State 2017


Procedural Generation

Generates a random song using cellular autonoma, and a normalized curve algorithm. Uses keys from the cycle of fourths to determine valid notes to use in the song.

Guided Generation

Generates a weighted probability map from pre-existing songs using their notes and compiling their probabilities, and then using that to create multiple notes
